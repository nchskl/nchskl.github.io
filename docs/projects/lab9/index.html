Лабораторная работа №9: Реализация CRUD для валют с использованием SQLite и архитектуры MVC

Цель работы

Основана на лабе 8. Целью данной лабораторной работы является реализация CRUD-операций (Create, Read, Update, Delete) для сущностей бизнес-логики приложения. В ходе работы изучаются основные принципы работы с базой данных SQLite в памяти (:memory:) через модуль sqlite3. Также будет освоена работа с архитектурой MVC, созданием контроллеров для взаимодействия с базой данных и рендеринга страниц, а также тестированием функционала с использованием unittest.mock.

Задачи работы:

  1. Реализовать операции CRUD для сущности Currency.
  2. Настроить взаимодействие с базой данных SQLite в памяти.
  3. Использовать параметры для защиты от SQL-инъекций.
  4. Понять принципы работы с первичными и внешними ключами.
  5. Реализовать контроллеры для работы с БД и рендеринга страниц.
  6. Создать роутер для обработки GET-запросов.
  7. Освоить архитектуру MVC, разделяя бизнес-логику и рендеринг данных.

Описание проекта

Проект реализует систему для работы с валютами и пользователями. Пользователи могут подписываться на валюты. Осуществляются операции добавления, отображения, обновления и удаления валют. Проект использует SQLite и архитектуру MVC для разделения бизнес-логики и рендеринга страниц.
